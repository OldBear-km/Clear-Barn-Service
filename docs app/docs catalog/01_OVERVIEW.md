# Catalog Service

## Назначение

Catalog — модуль хранения всех сущностей продукции организации.

Он является центральным источником данных для:

- печати этикеток
- складского учета
- рецептур
- меню
- закупок
- аналитики

Catalog работает на уровне **организации**, а не филиала.

Все сущности каталога принадлежат `organization_id`.

—

## Основные категории

Catalog содержит 4 типа сущностей:

### Product
Готовая продукция производства.

Пример:
- Круассан миндальный
- Сэндвич с тунцом

Используется для печати **продуктовых этикеток**.

—

### Goods
Товар для продажи.

Пример:
- шоколад
- печенье
- напитки

Особенность:
`годен до` может вводиться вручную при печати.

—

### Ingredient
Сырье кухни.

Пример:
- молоко
- сливки
- мука

Срок годности считается **после вскрытия**.

—

### SemiFinished
Полуфабрикаты кухни.

Пример:
- карамельный сироп
- крем
- тесто

Срок годности считается **от даты производства**.

—

## Связанные сущности

### Manufacturer

Производитель товаров и ингредиентов.

Используется для:
- печати этикеток товаров
